# 动态JSON下载指定字段的数据工具-DynamicJsonDownloader

主要解析JSON，批量下载视频和图片
> json格式示例为：
> [
    {
        "video_id": "1",
        "video_type": "1",
         "video_play_url": "https://test.com/1.mp4"
    }]  

格式是该格式，里面字段可随意。  

> [
    {
        "a_id": "1",
        "b_type": "1",
         "c_play_url": "https://test.com/1.mp4"
    }]
